## Exit criteria for Milestone 0

A new contributor can:

1. Clone the repo
2. `pnpm install`
3. `pnpm dev` and see all four stub apps run
4. `anchor build` and see the program compile
5. Read `docs/README.md` and find any spec section in ≤30 seconds

## Hand-off to Milestone 1

When all checkboxes above are ✅ and CI is green on `main`, the orchestrator advances to `tasks/milestone-1-document-pipeline.md` (to be generated by Planner).
